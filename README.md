# AirTicketBookingSystem

#### 介绍
JAVAWEB 期末项目 航空售票系统

#### 软件架构
通过MVC作为基础架构
采用Servlet + JSP + Jetty

#### 使用说明
导入IDEA使用Jetty容器运行
项目中 JS CSS Plugin 等 没用上传 采用Admin LTE 可自行导入

#### 参与贡献
UI界面采用 [Admin LTE](https://adminlte.io/)
